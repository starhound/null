# MCP Server Catalog

Null Terminal includes a curated catalog of **130+ MCP servers** across 22 categories. These servers extend AI capabilities through the [Model Context Protocol](https://modelcontextprotocol.io/).

Browse and install servers directly from Null Terminal:

```bash
/mcp catalog
```

## Getting Started

### Install from Catalog

The fastest way to add an MCP server:

```bash
/mcp catalog
```

This opens an interactive browser where you can search, browse by category, and install with one click.

### Manual Configuration

Add servers to `~/.null/mcp.json`:

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "your-token"
      },
      "enabled": true
    }
  }
}
```

### Commands

| Command | Action |
|---------|--------|
| `/mcp catalog` | Browse and install servers |
| `/mcp list` | Show configured servers |
| `/mcp add` | Add server manually |
| `/mcp enable <name>` | Enable a server |
| `/mcp disable <name>` | Disable a server |
| `/mcp remove <name>` | Remove a server |
| `/mcp tools` | List all available tools |
